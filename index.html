Mobile & Web Development

Hear Me App

React Native - Node JS - MongoDB - Heroku

Hear Me is an app that was developed during the last two weeks of the coding bootcamp I attended. In a team of three, we created this app to let users know about upcoming concerts and performers in their city. The app was built in React Native with NodeJS, used SongKick’s API, our own API hosted on Heroku and a NoSQL Database.

GitHub repository

Calculator App

React Native (iOS & Android)

This Calculator App was a great way to learn React Native and more JavaScript. The interesting thing about this calculator is that I used React Native’s capabilities to an extent where I made the app completely functional on both Android and iOS devices. My goal is to now integrate this calculator in an educational app.

GitHub repository

Movies Website

React JS - Live on Heroku - Mobile responsive

This project allowed me to learn more about React JS, API get requests as well as deployments on Heroku. This webpage, made entirely with React JS, allows users to browse popular, upcoming and top rated movies. Get requests to the The Movie Database API were made to provide the data.

GitHub repository

AR Drive

Swift - ARKit

AR Drive is an Augmented Reality iOS application, written in Swift, enabling users to replace horizontal surfaces with a road texture on which they can drop a vehicle. Using the iPhone's accelerometer, the wheels will steer either left or right. To drive the vehicle, simply tap on the screen.

Watch Demo

AR Draw

Swift - AR Kit

AR Draw is an Augmented Reality iOS app allowing users to draw in the real world! When detecting enough tracking points, users can use a brush to draw in any physical space. The erase button will reset the camera view so that new drawings can be done.

Watch Demo

AR Art Book

Unity - Maya - Vuforia SDK

AR Art Book augments an art book cover by opening it up and revealing a 3D Mona Lisa painting. In order to realize this project, I had to use a variety of tools and softwares: Vuforia for tracking, Maya for the 3D animation and Unity to build the project for iOS.
